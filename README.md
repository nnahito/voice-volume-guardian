# Sound Guard
マイクに一定音量以上の入力があると、wavファイルを再生します

## 使い方
### （0）Python
自分で入れてください。
scoopとかでサクッと入れると楽。

### （1）venvの準備
```
$ python3 -m venv venv
$ venv\Scripts\activate

## macは source venv/bin/activate
```

### （2）必要モジュールのダウンロード
```
$ pip install -r requirements.txt
```

### （3）起動
```
$ python main.py
```
